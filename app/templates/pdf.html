ARQUIVOS PDF

O Adobe Acrobat Reader é o visualizador de arquivos multiplataforma usado para ler os arquivos de extensão PDF.
Caso não tenha instalado, faça o download e instale o Acrobat Reader para ler os relatórios.

  • Salve o arquivo executável (com extensão .exe) em uma pasta ou diretório temporário em seu computador.
  • Execute este arquivo clicando com o mouse duas vezes sobre ele.
  • Siga as instruções de instalação.
  • Com o programa Adobe Acrobat Reader instalado em seu computador, você já pode ler arquivos em pdf.
  • Como os navegadores estão visualizando os arquivos pdf é provável que ao clicar em um link para arquivos pdf você já os visualize no próprio navegador. Caso queira baixar para o próprio micro, clique com o botão direito do mouse no link e salve os arquivo localmente. Dai dê um duplo clique para abrí-los.
